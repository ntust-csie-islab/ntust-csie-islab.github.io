How do I create fill-out forms?

Contributors:
  • boutell boutell@boutell.com
Forms are generally NOT USEFUL until you (or someone else) writes a CGI program to interpret the results on the server side! For more information, see How do I generate web pages from a program (CGI)?

"I know how to write CGI programs, I just don't know how to write forms in HTML."

OK, see the w3 consortium pages <URL:http://www.w3.org/hypertext/WWW/MarkUp/html-spec/> for the HTML specification. For a less technical approach, consult virtually any book about HTML for information about forms. A good HTML reference guide is a worthwhile investment.

See the section on email and comment forms for a simple solution to the most commonly desired forms.
