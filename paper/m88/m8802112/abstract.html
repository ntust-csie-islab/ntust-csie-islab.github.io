運用灰色關聯分析建構之個人化課程網頁瀏覽

摘要

當使用者想在網際網路上找尋所需資訊的同時，由於不同的使用者有不同的喜好及需要， 所以提供個人化的網頁已經不再只是附加價值的服務，而是大型資訊系統必要具備的一項 功能。並且，能夠即時提供使用者個人化資訊及建議，更能提高整個建議機制的效能，尤 其應用在電子商務上，能創造更多的商機。在本篇論文裡，我們提出了架構一個在遠距教 學系統上，可以線上即時分析使用者瀏覽行為及瀏覽興趣的方法，利用加權灰色關聯分 析 (Weighted Grey Relational Analysis) 來分析使用者的喜好資訊，進而提供使用者個 人化的課程瀏覽資訊。使用者可以選擇使用查詢功能或者是瀏覽課程主題。系統藉由觀察 使用者的瀏覽行為來取得有關使用者的資訊，而使用者可以正常的使用而不會察覺系統的 這個動作。我們利用使用者點選過的文章內容作分析，並考慮使用者瀏覽文章的時間及文 章的長度來，最後得到使用者瀏覽喜好的分析結果。實驗顯示，使用者瀏覽的時間越長， 能表示對所瀏覽的文章越有興趣。這個分析方法計算方式簡單，所以容易達到線上即時分 析的效果，並且使用者的紀錄也可以動態的修改，進而提供即時的符合使用者興趣的建議 及資訊。

Personalized Courseware Navigation Using Grey Relational Anlaysis

Abstract

Since individual users have difference needs and preferences as they seek information, Web personalization has quickly changed from an added-value property to a necessary service on large information presentation.　In this thesis a personalized information recommending system with online preference analysis, termed as the Interface Agent, is developed on a distance-learning environment called as the Coursebot.　Users can both browse and search information on the interface of the Coursebot.　And our system provides appropriate course materials in rank which reflect user’s interests after a user gives requests. We propose an analyzing measure, which combines the typical grey relational analysis and implicit rating, to analyze user’s interests from the content of documents and user’s browsing behavior.　With the properties of this algorithm such as low computational complexity and ease of adding prior knowledge, it can ease online personalized analysis.　The profiles are revised dynamically so as to provide personalized information reflected user’s interests efficiently after users visit page by page.　Moreover, profiles do not extend continually which results in lower memory storage required for profiles.
