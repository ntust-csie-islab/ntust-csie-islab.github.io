一個有效率的多類分離模糊資訊分類器設計

摘 要

 本篇論文提出一個以類神經網路為結構，範例 (exemplar) 為基礎且具有處理分離 模糊資訊能力之分類器。在這個網路中, 隱藏層 (hidden layer) 的範例節點 (exemplar node) 表現出在特徵空間中的分離模糊資訊和例外情形。本分類器能夠依照 分佈在特徵空間的分離樣本群自動地產生和調整範例。由於這些範例節點的數目並不是 固定的，所以這些節點會形成接近最佳判斷區域 (near-optimal decision regions) 來近似輸入樣本的分佈。此外，分佈於特徵空間的例外情形可由巢狀 (nested) 範例 節點來表示。本分類器之訓練法則分成建立、擴張和合併範例節點等程序，所有程序皆 在一個單週期 (single epoch) 內完成，而且具有線上學習 (on-line learning) 的能 力，此訓練策略可以減少記憶體的需求，使非線性分類 (nonlinear classification) 的過程加速，亦可迅速處理分離模糊資訊和例外情形。本分類器是採用LR型式的梯形模 糊區間 (trapezoidal fuzzy interval LR-type) 表示模糊資訊，所以只需少量的模糊 計算。經由IRIS和KBE的實驗結果顯示，此分類器能產生適當的範例節點來表現分佈在 樣本空間的樣本群，且對於分離資料庫亦有不錯的的辨識率。

An Efficient Exemplar-Based Fuzzy Neural Network Model for Disjunctive Fuzzy Inputs

Abstract

 This paper presents an efficient exemplar-based fuzzy neural network model that learns disjunctive fuzzy information in the feature space. The exemplar nodes in the hidden layer represent disjunctive fuzzy information and exceptions in the feature space. This classifier can automatically generate and refine exemplars for distinct clusters. The number of these exemplars is not fixed, and thus the exemplars can form near-optimal decision regions to meet the distribution of input patterns. The exceptions distributed over pattern clusters can be represented by nested exemplars. The training algorithm of this proposed classifier consists of creating, expanding and merging exemplars. All these operations are finished in a single epoch. Also the on-line learning is supplied. Such a training strategy can reduce the memory requirement and speed up the process of nonlinear classification. In addition, the disjunctive fuzzy information and exceptions are handled easily. Since input patterns and all the nodes in the proposed classifier are represented in trapezoidal fuzzy interval LR-type, the computational load of this classifier is lightened. The experimental results manifest that this model can determine appropriate number of exemplar nodes and obtain good performance in some real examples such as IRIS and KBE.
