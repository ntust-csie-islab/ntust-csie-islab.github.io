經由錯誤預測提升系統一般化能力

摘　要

本篇論文藉由使用一個錯誤預測修復系統來改進原始系統一般化的能力。 藉由建立一個專門預測原始系統錯誤的錯誤預測系統，原始系統的錯誤 可以利用此錯誤預測系統來修正。因為此錯誤預測系統是用來預測原始 系統的錯誤，所以此錯誤預測系統的訓練資料其輸出部份應該是原始系 統所產生的錯誤值。然而一般測試資料並不具正確輸出值，所以其錯誤 值也無從得知。為了要產生錯誤預測系統的訓練資料，我們建議使用bootstrap方法， 從原始系統的訓練資料中預測其可能發生的錯誤。為了改進bootstrap方法 對運算時間的效率，我們提出一個新的方法 central bootstrap。 Central bootstrap 是藉由分析bootstrap取樣集合和bootstrap 預測結果的關係， 推論假如bootstrap取樣集合的每個取樣點都位於兩個鄰近的原始訓練資料 的中心。那利用這個bootstrap取樣集合所得到的預測值應該不錯。 這個bootstrap取樣集合就命名為central bootstrap取樣集合，利用這個取樣 集合做預測的bootstrap方法，就稱做central bootstrap方法。我們的實驗是 應用函數趨近問題來做測試。實驗中錯誤預測修復系統分別利用 cross-validation,bootstrap 和 central bootstrap三種方法來建構。實驗結果 顯示由central bootstrap建構的錯誤預測修復系統，通常錯誤率最低並且 所需的計算量也最低。由bootstrap建構的錯誤預測修復系統，在不同次 的執行中彼此結果差異量最小，並且都能保證修正後的輸出一定會比原始系統輸出好。

Generalization Improvement by Error Prediction

Abstract

In this thesis, an error prediction reviser (EPR) is introduced to improve the generalization ability of the original model. By constructing an error predictor (EP), which is used to predict the testing error of the original model, the testing error of the original model can be revised by the output of EP. Because EP is used to predict the testing error of the original model, the output of the training data of EP must be the testing error of the original model. However, the testing error of the original model is unknown because the desired output of the testing data is unknown. In order to generate the training data set of EP, we propose to use bootstrap method to predict the testing error of the original model from the original training data set. To improve the efficiency of bootstrap method, we also present central bootstrap method. By analyzing the relationship between the bootstrap sample set and the prediction using bootstrap method, we infer that if each bootstrap sample is in the center of the two neighboring original training data, the prediction will be good. This bootstrap sample set is named central bootstrap sample set, and bootstrap method with central bootstrap sample set is named central bootstrap method. In our experiments, a function approximation problem is applied, and EPR is individually constructed by cross-validation, bootstrap and central bootstrap methods. The results show that EPR with central bootstrap has the lowest computing load and the smallest testing error in average. Also, EPR with bootstrap has the smallest deviation of the testing error, and it is always guaranteed to improve the testing error of the original model.
