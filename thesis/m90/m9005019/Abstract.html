利用負向資料萃取配合多類別支援向量機之網頁分類器

摘要

隨著網站數量的大量增加，網頁分類已經成為一個重要的研究主題，然而傳統分類方法的效能並不顯著。在最近幾年，支援向量機在文件分類的領域中展現了極佳的分類效能，支援向量機在原始的設計上是一個二元的分類器，因此需要搭配多類別的方法使用，值得注意的是，一個網頁中可能包含了多個類別的內容在裡面，一對多方法可以適合這個特色搭配支援向量機應用在網頁分類上。主要的問題是在一對多的架構下，負向資料的數目會太過龐大而影響到訓練的過程，在支援向量機中，過多的負向資料會大大增加訓練的時間。 在這篇論文中，我們提出了負向資料萃取的方法，能夠大幅降低支援向量機中負向資料的數目，我們使用文件出現頻率和餘弦定理來挑出具有代表性的負向資料，只有被選取的資料被加入到支援向量機的訓練中，因此，負向資料的數目能夠有效的降低，能夠大幅改善訓練時間過久的問題。實驗的結果顯示，使用我們的方法後，分類的準確率並不會因為負向資料數目的減少而降低，證明在一對多的架構下，負向資料萃取方法在縮短支援向量機的訓練時間中能夠有非常傑出的表現。

Abstract

With the rapid growth of web sites, web pages classification has already become an important research field. However, the effect of traditional classification method is limited. In the recent years, Support Vector Machine (SVM) has been shown to have great performance in document classification field. SVM is a binary class classification tool originally; therefore, some multi-class method should be applied on it. Notably, a web page could have multiple classes and One-against-All method can be used in web page classification with SVM. The main problem is that the negative data will be too huge to train in One-against-all structure. Large negative data will increase the training time of SVM. In this thesis, a Negative Data Selection method is proposed to reduce the huge amount of negative data in SVM. Document Frequency measure and Cosine measure are used to select the representative negative data. Only the selected negative data are used for training. Thus, the number of negative data can be reduced. The situation of large training time with SVM will be improved. The experimental result reveals that the accuracy of classification will not decrease with part of negative data for training. Also the Negative Data Selection method has excellent performance in reducing training time of SVM with One-against-All structure.
